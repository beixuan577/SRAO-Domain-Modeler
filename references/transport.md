# 交通/隧道领域模型

## 核心概念词典

| 概念 | 类型 | 定义 | 关键属性 |
|------|------|------|----------|
| 隧道段 | 实体 | 隧道的区段划分 | 段号、桩号、围岩等级、埋深 |
| 点云 | 实体 | 三维扫描数据 | 采集时间、分辨率、坐标系 |
| 位移场 | 实体 | 变形位移分布 | 方向、量值、变化率 |
| 热力图 | 实体 | 异常分布可视化 | 区域、等级、时间趋势 |
| 预警事件 | 事件 | 超阈值报警 | 等级、位置、持续时间 |
| 消防巡检 | 实体 | 机器人消防检查 | 路线、热成像、设施状态 |
| 数字孪生 | 实体 | 实时虚拟映射 | 模型、数据流、可视化 |

## 实体关系

```
[隧道段] --1:N--> [点云]
[点云] --配准--> [位移场]
[位移场] --生成--> [热力图]
[热力图] --触发--> [预警事件]
[隧道段] --1:N--> [消防巡检]
[消防巡检] --输入--> [数字孪生]
[预警事件] --输入--> [数字孪生]
```

## 工作流模板

### WF-TRN-001: 围岩失稳监测

```yaml
trigger: "周期性采集完成"
steps:
  - id: s1
    name: 多源数据融合
    role: 融合感知Agent
    input: 激光雷达 + 多目相机
    output: 融合点云
    next: [s2]
  - id: s2
    name: 点云配准
    role: 配准Agent
    input: 当期点云 + 基准点云
    output: 配准后点云对
    next: [s3]
  - id: s3
    name: 位移场计算
    role: 位移计算Agent
    input: 配准点云对
    output: 位移场 + 变形量
    next: [s4]
  - id: s4
    name: 热力图生成
    role: 可视化Agent
    input: 位移场
    output: 变形热力图
    next: [s5]
  - id: s5
    name: 时段对比
    role: 时序分析Agent
    input: 当期 + 历史热力图
    output: 变化趋势 + 预警
```

### WF-TRN-002: 消防机器人巡检

```yaml
trigger: "定时 / 温度异常"
steps:
  - id: s1
    name: 热成像异常检测
    role: 热成像Agent
    input: 红外相机数据
    output: 高温区域标记
    next: [s3]
  - id: s2
    name: 设施缺损识别
    role: 目标检测Agent
    input: 可见光相机
    output: 消防设施状态
    next: [s3]
  - id: s3
    name: 人员定位
    role: 多模态跟踪Agent
    input: 热成像 + 可见光
    output: 人员位置列表
    next: [s4]
  - id: s4
    name: 下发指令
    role: 指令Agent
    input: 异常汇总 + 人员位置
    output: 巡检报告 + 应急指令
```

## 典型KPI

| KPI | 基线 | 目标 |
|-----|------|------|
| 监测周期 | 3天 | 10分钟 |
| 微小变形检出 | 无法发现 | ≥0.5mm |
| 消防巡检覆盖率 | 60% | 100% |
| 误报率 | 40% | <5% |
