# 农业领域模型

## 核心概念词典

| 概念 | 类型 | 定义 | 关键属性 |
|------|------|------|----------|
| 地块 | 实体 | 种植区域 | 地块ID、面积、作物类型、土壤类型 |
| 作物 | 实体 | 种植的农作物 | 品种、生长期、预计产量 |
| 土壤数据 | 实体 | 传感器采集的土壤参数 | 湿度、pH、氮磷钾含量 |
| 病虫害 | 实体 | 作物病害/虫害 | 类型、严重度、面积、扩散方向 |
| 气象数据 | 实体 | 气象站/预报数据 | 温度、降水、湿度、风速 |
| 施药处方 | 实体 | 精准施药方案 | 药剂、浓度、用量、区域 |
| 产量预测 | 实体 | 收成预估 | 预测值、置信度、时间 |

## 实体关系

```
[地块] --1:1--> [作物]
[地块] --1:N--> [土壤数据]
[地块] --1:N--> [病虫害]
[气象数据] --影响--> [病虫害]
[气象数据] --影响--> [产量预测]
[土壤数据] --输入--> [施药处方]
[病虫害] --输入--> [施药处方]
```

## 工作流模板

### WF-AGR-001: 病虫害预警

```yaml
trigger: "无人机巡检周期 / 气象异常"
steps:
  - id: s1
    name: 多光谱采集
    role: 光谱分析Agent
    input: 无人机多光谱影像
    output: 植被指数异常区
    next: [s2]
  - id: s2
    name: 识别病虫害
    role: 病虫害识别Agent
    input: 异常区高分辨率图像
    output: 病虫害类型 + 严重度
    next: [s3]
  - id: s3
    name: 扩散模型预测
    role: 扩散仿真Agent
    input: 病虫害 + 气象 + 地块
    output: 3/7/14天扩散预测
    next: [s4]
  - id: s4
    name: 生成施药建议
    role: 施药处方Agent
    input: 病虫害 + 扩散预测
    output: 施药处方
```

### WF-AGR-002: 水肥一体化

```yaml
trigger: "定时 / 土壤阈值触发"
steps:
  - id: s1
    name: 采集土壤数据
    role: 土壤数据Agent
    input: 传感器网络
    output: 土壤参数矩阵
    next: [s3]
  - id: s2
    name: 获取气象预测
    role: 气象Agent
    input: 地块位置
    output: 7天气象预报
    next: [s3]
  - id: s3
    name: 作物模型计算
    role: 作物生长模型Agent
    input: 土壤 + 气象 + 作物参数
    output: 水肥需求量
    next: [s4]
  - id: s4
    name: 控制灌溉阀门
    role: 灌溉控制Agent
    input: 水肥需求量
    output: 阀门开度指令
```
